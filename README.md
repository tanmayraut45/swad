# Swad Restaurant

A premium, elegant restaurant portfolio website for Swad Restaurant and Udappi Center.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) to view the website.

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Radix UI / shadcn/ui
