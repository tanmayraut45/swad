# **App Name**: Maha Zaika

## Core Features:

- Landing Page with Generative Hero: Showcase a dynamic hero section on the landing page. Use generative AI to adapt the background image to the current season, reflecting either summer or winter themes in Maharashtra.
- Restaurant Information: Display a detailed 'About Us' section highlighting the restaurant's history as a family-owned establishment, emphasizing its dedication to authentic Maharashtrian cuisine, enhanced with cultural design cues.
- Digital Menu: Present an interactive digital menu featuring all food and drink items. Include names, descriptions (with Marathi translations), prices, and attractive images of each dish. Also offer an option to download the menu as a PDF.
- Interactive Map Integration: Integrate an interactive Google Map pinpointing the restaurant's exact location using provided coordinates, along with a custom-designed pin, and a 'Get Directions' button. The map should visually align with the restaurant's branding colors for consistency.
- Gallery Showcase: Build a visually appealing gallery displaying high-quality images of the restaurant's ambiance, dishes, and customer experiences. Use a masonry layout to keep the images well-organized and make the gallery appealing.
- Online Booking and Contact: Implement a straightforward booking form for reservations, allowing users to submit their name, phone number, and preferred date and time. Provide direct call and WhatsApp buttons, display opening hours, and provide addresses in both Marathi and English.
- Multilingual Tool: Offer seamless toggling between Marathi and English to ensure accessibility for all users, which the LLM must check to determine correct behavior when switching languages.

## Style Guidelines:

- Primary color: Deep maroon (#800000) to evoke tradition and richness.
- Background color: Ivory (#FFFAF0), offering a clean, elegant backdrop that enhances readability and overall visual appeal.
- Accent color: Emerald green (#50C878), providing a vibrant contrast and drawing attention to key interactive elements, enriching the visual hierarchy and user experience.
- Headline font: 'Playfair', a modern serif with an elegant, high-end feel. Body font: 'PT Sans', a humanist sans-serif that combines a modern look with warmth; providing excellent readability for all text elements.
- Use elegant, custom-designed icons reflecting Maharashtrian culture and cuisine.
- Employ a cinematic layout with full-screen elements, parallax scrolling, and smooth transitions to create an immersive user experience.
- Incorporate subtle micro-animations and hover effects to enhance interactivity and visual appeal, without overwhelming the user.
